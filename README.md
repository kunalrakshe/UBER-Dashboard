This repository includes a comprehensive end-to-end Power BI analysis of Uber's business performance. The project converts unprocessed ride-level data into useful insights about operations, revenue, consumer behavior, geographic trends, and strategic suggestions.


### 📌 Project Synopsis
Uber is a global mobility platform that uses real-time technology to connect drivers and passengers.Using reliable data modeling and visualization, this project seeks to examine trip performance, cancellation patterns, revenue trends, and operational effectiveness.

### 🔍 Insights Summary
#### 1. Operational Performance
- [x] 93K completed rides → Strong reliability
- [x] 57K lost bookings → 38% revenue loss
- [x] 37K cancellations → requires operational fixes

#### 2. Revenue
- [x] Total Revenue: ₹52M
- [x] Avg Fare: ₹345.64
- [x] UPI contributes 44% of total revenue
- [x] Highest revenue from: Auto (₹13M)

      
####  3. Demand Patterns
- [x] Peak demand: 6 PM – 9 PM
- [x] Consistent weekly demand
- [x] Distance dipped sharply in February

      
####  4. Customer Behavior
- [x] 104K customers
- [x] Driver rating lower than customer rating
- [x] Major cancellation reason: Blank entries (29,942)

      
####  5. Vehicle Insights
- [x] Auto & Bike dominate
- [x] Premium sedan & XL underutilized


#### 6. Growth Trends    
- [x] Volatile growth range: -12.3% to +15%
- [x] Negative dips in Feb, Aug, Nov, Dec

      
      
### 🧭 Strategic Recommendations
#### Revenue & Growth
- [x] Reduce lost bookings using surge optimization
- [x] Promote premium vehicles
- [x] Stabilize revenue in low-demand months

#### Operational Improvements
- [x] Improve driver allocation during peak hours
- [x] Enforce stricter cancellation rules
- [x] Fix blank cancellation reason issue

#### Customer Experience
- [x] Close rating gap through driver training
- [x] Loyalty program for regular riders
- [x] Onboarding offers for new riders

#### Fleet & Geography
- [x] Add vehicles at high-demand hubs (Saket, Barakhamb)
- [x] Build dedicated corridors between heavy pickup-drop pairs
